# basename

#### 简介
basename 

#### install
```bash
env GOPATH=`pwd` go get -u github.com/guonaihong/basename
```

#### 命令行选项
```console
Usage of basename:
  -a, --multiple
    	support multiple arguments and treat each as a NAME
  -s, --suffix string
    	remove a trailing SUFFIX; implies -a
  -z, --zero
    	end each output line with NUL, not newline
```
